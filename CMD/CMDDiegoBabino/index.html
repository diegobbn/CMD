»About

This project is an experimental showcase created by Diego Babino (me) for showcasing my portfolio in an unusual but original way by simulating a command line interface. It's nothing more than the usual hidden elements triggered by some JS to show/hide each section of content, based on input values.

This project has been specifically designed for my application to the CEPV in the “formation supérieure en communication visuelle option photographie.” I believe it reflects my innovative approach to visual communication and my commitment to exploring new ways of presenting my artistic vision.

»Who am I ?

Using it is pretty straight forward. Create several(as much as you need) divs or «section» elements each with it's unique id to hold your content. Make sure to have an error section (id="error") with some information to warn the user when he/she mistypes a command or if there is no section with that name, think of it as a 404 page.

A section with id="help" is also required so you can list all the available sections and let your users know what words to write to access that particular content.

The input field must be a standalone element and must not be placed inside any of the content divs or sections, and make sure you give it a classname "command", if you want to name it anything else you need to edit the app.js and change all instances of .command in the script to match your chosen classname.

In short, just view the source code of this demo, copy/modify it to your liking making sure you include the error and help sections.

Ushi is a project that explores whether AI has a place in the world of music. As a photographer/Web designer with a passion for new technology and art, I seek to push creative boundaries through innovation.

This project is an experiment and an extension of my universe, where I have developed an AI that autonomously generates sounds, composing and producing music without human intervention. The entire process is fully automated—from sound design to arrangement—allowing the AI to create unique compositions. My role is to curate and mix the best pieces, shaping the final tracks while letting the AI be the true architect of the sound.
